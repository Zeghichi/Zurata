# Zurata

Zurata is a lightweight Python utility project. It provides a simple starting point for building Python-based tools and scripts.

## Requirements

- Python 3.8 or higher

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Zeghichi/Zurata.git
   cd Zurata
   ```

2. (Optional) Create and activate a virtual environment:

   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

## Usage

Run the main script directly:

```bash
python hhoga.py
```

You can also import it as a module in your own code:

```python
import hhoga
```
